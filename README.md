# analisis-ventas-ecommerce
Proyecto de análisis de ventas de un e-commerce de principio a fin utilizando Python, SQL y Power BI para transformar datos en información útil para el negocio.
## Cuadro de Mando Interactivo (Looker Studio)

Se ha diseñado un dashboard interactivo para analizar el rendimiento comercial, el comportamiento de las ventas mensuales, la distribución regional y las preferencias de pago de los clientes.

Puedes acceder y filtrar los datos en tiempo real a través del siguiente enlace:
